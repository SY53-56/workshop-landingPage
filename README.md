AI & Robotics Workshop Landing Page

A modern and responsive landing page built to promote an AI & Robotics Workshop. The platform provides workshop information, learning outcomes, schedules, and a registration form for participants.

Features:

* Responsive UI for desktop and mobile devices
* Hero section with immersive background video
* Workshop details and learning objectives
* Registration form with backend API integration
* Data storage using MongoDB Atlas
* Full-stack deployment using Vercel and Render

Tech Stack:

* Frontend: React, TypeScript, Vite, Tailwind CSS
* Backend: Node.js, Express.js
* Database: MongoDB Atlas
* Deployment: Vercel (Frontend), Render (Backend)

The project demonstrates full-stack web development, API integration, database connectivity, responsive design, and cloud deployment.
